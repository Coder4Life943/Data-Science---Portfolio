# Cloud Computing Migration – Feasibility and Security Analysis (2025)

## Module
Virtualisation and Cloud Computing

## Result
First Class, 71%

## Project Overview
This project analyses the feasibility, challenges, and security considerations involved in migrating an organisation’s IT infrastructure to cloud computing environments.

The report evaluates how cloud technologies can improve scalability, flexibility, and operational efficiency, while also addressing risks related to security, legal compliance, and legacy system compatibility.

The scenario focuses on advising a company on the best approach to transition from traditional IT infrastructure to cloud-based solutions.

## Cloud Concepts Covered

- Infrastructure as a Service (IaaS)
- Platform as a Service (PaaS)
- Software as a Service (SaaS)
- Virtualisation technologies
- Cloud scalability
- Shared responsibility model
- Hybrid cloud strategies

## Key Areas of Analysis

### Migration Feasibility
Evaluation of whether existing business systems can be successfully migrated to cloud environments, considering:

- compatibility of legacy systems
- cost implications
- operational disruption risks
- technical complexity

### Security and Risk Assessment

Analysis of major cloud security concerns including:

- data breaches
- data loss prevention
- service provider failure risks
- access control and authentication
- GDPR compliance requirements
- disaster recovery strategies

### Legacy Systems and Virtualisation

Assessment of how virtual machines can be used to support legacy systems during migration, allowing organisations to modernise infrastructure gradually while maintaining business continuity.
