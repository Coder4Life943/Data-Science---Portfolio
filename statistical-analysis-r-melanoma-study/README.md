## Project Overview
This project involved performing an exploratory data analysis using R on a real medical dataset related to melanoma survival outcomes.

The aim was to apply statistical techniques to understand relationships between tumour thickness, patient age, gender, and survival time.

The analysis demonstrates the use of statistical modelling techniques commonly used in data science and healthcare analytics.

## Dataset
Melanoma survival dataset containing patient measurements including:

- survival time
- tumour thickness
- age
- gender
- ulceration status
- year of surgery

The dataset was analysed to identify statistically significant relationships between variables linked to patient outcomes.

## Statistical Techniques Used

- Descriptive statistics
- Histograms
- Boxplots
- Correlation analysis
- Linear regression
- Hypothesis testing
- Welch two-sample t-test
- QQ plots
- Data visualisation in R

## Key Findings

- Tumour thickness is a statistically significant predictor of survival time
- Older patients tend to show lower survival times
- Male patients showed slightly lower survival times compared to females
- Tumour thickness increases slightly with age
- Statistical tests confirmed significant relationships between key medical variables

The analysis demonstrates understanding of statistical inference and interpretation of real-world health data.
