# Data Visualisation in Tableau – NBA Scoring Analysis (2025)

## Module
Data Visualisation

## Project Overview
This project focuses on using Tableau to analyse sports performance data and communicate insights through professional visualisations.

The objective of the project was to investigate whether Michael Jordan is the greatest scorer in NBA history using statistical evidence and data visualisation techniques.

The project demonstrates how data visualisation can be used to transform raw data into meaningful insights that support decision-making.

## Dataset

NBA player statistics dataset containing performance metrics such as:

- Points per Game (PpG)
- Games Played (G)
- Field Goal Percentage (FgP)
- Player performance across different seasons

Data was collected from Kaggle and analysed using Tableau dashboards.

## Visualisation Approach

The analysis focused on comparing top NBA scoring seasons using multiple metrics.

Custom calculated fields were created in Tableau to evaluate scoring efficiency and consistency.

Key comparisons included:

- scoring volume vs efficiency
- performance consistency across seasons
- comparison of historical top scorers
- evaluation of different scoring metrics

Visualisations were designed to clearly communicate trends and support analytical conclusions.
