## Project Overview
This project applies machine learning algorithms to real-world datasets for both regression and classification tasks.

The objective was to build models, compare their performance, and evaluate which algorithms produced the most reliable predictions.

## Regression Task
Two regression algorithms were implemented and compared:

- Random Forest Regressor
- Support Vector Regression (SVR)

The models were evaluated using R² scores across multiple random states to test consistency and performance.

Results showed strong predictive performance, with Random Forest achieving approximately 0.97 R² and SVR achieving values close to 1.00, indicating possible overfitting.

## Classification Task
Two classification algorithms were implemented and compared:

- Gaussian Naive Bayes (GaussianNB)
- Support Vector Classifier (SVC)

Models were evaluated using:

- Precision
- Recall

The analysis considered the importance of precision in medical prediction scenarios such as heart disease classification.
