# Computer Networks and Internetworking (2024)

## Module
Networks and the Internet

## Result
First Class, 75%, 100%

## Project Overview
This project explores key concepts in computer networking and internetworking, combining theoretical research with practical network configuration tasks.

The work demonstrates understanding of modern networking technologies, including the evolution of mobile communication systems from 1G to 5G, and future developments such as 6G and 7G.

Practical activities were completed using Cisco Packet Tracer to simulate and evaluate network designs.

## Research Topic

Impact of 5G, 6G and 7G on the future of internetworking.

The research examines how future network technologies will improve:

- Speed and bandwidth
- Latency reduction
- Reliability of communication systems
- Internet of Things (IoT) connectivity
- Autonomous systems and smart cities
- Edge computing capabilities

## Practical Networking Skills (Cisco Packet Tracer)

Weekly labs demonstrated configuration and testing of network devices, including:

- Router configuration using CLI
- Switch configuration
- VLAN configuration
- Trunking configuration
- IP addressing and subnetting
- Telnet configuration
- Static routing
- Network connectivity testing using ping
- VTP (VLAN Trunking Protocol) configuration
- Inter-VLAN routing

## Skills Demonstrated

- Network configuration and troubleshooting
- CLI (Command Line Interface) usage
- VLAN and routing concepts

- Technical documentation skills
- Research and academic writing
